动态规划（Dynamic Programing，DP）是一种在数学、计算机科学和经济学中使用的，通过将复杂问题分解为**更小的、重叠子问题**来解决优化问题的方法。核心思想是记住已经解决过的子问题的解，从而避免重复计算，提高算法效率。
# 一、核心思想和适用条件
动态规划需要满足两个核心特性：
1. 最优子结构：
	- 定义：一个问题的最优解包含其子问题的最优解。换言之，我们可以通过组合子问题的最优解构造原问题的最优解。
	- 举例：在寻找从A点到Z点的最短路径问题中，如果最短路径经过了M点，那么从A到M的路径也必然是A到M的最短路径。
2. 重叠子问题：
	- 定义：在问题的求解过程中，许多子问题的计算会被重复多次。动态规划正是通过存储这些子问题的解来避免重复计算。
	- 举例：在计算斐波那契数列 `Fib(n) = Fib(n-1) + Fib(n-2)` 时，计算 `Fib(5)` 需要 `Fib(4)` 和 `Fib(3)`，而计算 `Fib(4)` 又需要 `Fib(3)` 和 `Fib(2)`。`Fib(3)` 被重复计算了。
# 二、核心步骤
| 要素       | 说明                | 示例（斐波那契数列）                  |
| -------- | ----------------- | --------------------------- |
| **状态定义** | 用 `dp[i]` 表示子问题的解 | `dp[i]` = 第 `i` 个斐波那契数      |
| **状态转移** | 递推关系式（最关键步骤）      | `dp[i] = dp[i-1] + dp[i-2]` |
| **初始条件** | 最小子问题的解（边界值）      | `dp[0]=0`, `dp[1]=1`        |
| **递推顺序** | 确定计算顺序（自底向上或自顶向下） | `i` 从 2 到 `n` 循环计算          |
| **结果提取** | 从 `dp` 数组中获取最终解   | `dp[n]` 为结果                 |
# 三、实现方式
| 特性   | 自顶向下-记忆化搜索                                               | 自底向上-递推/表格法                                                  |
| ---- | -------------------------------------------------------- | ------------------------------------------------------------ |
| 思路   | 从目标问题开始，递归地分解成子问题。如果一个子问题已经被解决过，直接返回存储的结果；否则，计算它并将结果存起来。 | 从最小的子问题开始，迭代地计算并存储所有子问题的解，直到解决目标问题。                          |
| 实现方式 | 递归+备忘录                                                   | 循环+数组/表格                                                     |
| 优点   | - 实现通常更直观，接近人类的递归思考方式。  <br>- 只计算被需要（被递归调用到）的子问题。        | - 避免了递归的开销（如函数调用栈），通常效率更高。  <br>- 空间复杂度可能更优（例如可以使用“滚动数组”优化）。 |
| 缺点   | - 可能会因为递归深度过大而导致栈溢出。  <br>- 函数调用的额外开销可能导致性能略低。           | - 需要明确所有子问题的计算顺序。  <br>- 可能会计算一些最终用不上的子问题的解                  |
# 四、核心决策模式
1. **选/不选模型（最核心决策）**
	- 当前元素是否参与最优解构建
	- 状态转移通式：
		dp[i]=max(选择当前项的价值+子问题解，不选当前项的子问题解)
	- 经典应用：
		```python
打家劫舍
dp[i] = max(dp[i-1],             # 不抢第i家
            dp[i-2] + nums[i])   # 抢第i家

0/1背包
dp[w] = max(dp[w],                # 不选当前物品
            dp[w-wt] + val)       # 选当前物品
```
2. 多状态决策模型
	- 定义多个状态表示不同选择路径
	- 经典应用：
	```python
股票买卖（含冷冻期）
dp0[i] = max(dp0[i-1], dp1[i-1] + prices[i])  # 持有股票
dp1[i] = max(dp1[i-1], dp0[i-2] - prices[i])  # 未持有（冷冻期）
```
# 五、经典例题
## 1. 爬楼梯
- 题目链接：[[https://leetcode.cn/problems/climbing-stairs/description/]]
- 思考：
- 因为要解决的问题都是「从 0 爬到 i」，所以定义 dfs(i) 表示从 0 爬到 i 有多少种不同的方法。
- 分类讨论：
	- 如果最后一步爬了 1 个台阶，那么我们得先爬到 i−1，要解决的问题缩小成：从 0 爬到 i−1 有多少种不同的方法。
	- 如果最后一步爬了 2 个台阶，那么我们得先爬到 i−2，要解决的问题缩小成：从 0 爬到 i−2 有多少种不同的方法。
- 由于这两种方法是互相独立的（爬的台阶个数不同），所以根据加法原理，从 0 爬到 i 的方法数等于这两种方法数之和，即 dfs(i)=dfs(i−1)+dfs(i−2)
- 递归边界：dfs(0)=1, dfs(1)=1。从 0 爬到 0 有一种方法，即原地不动。从 0 爬到 1 有一种方法，即爬 1 个台阶。
- 递归入口：dfs(n)，也就是答案。
```python
class Solution:

def climbStairs(self, n: int) -> int:

f = [0] * (n+1)

if n == 1:

return 1

f[1] = 1

f[2] = 2

for i in range(3,n+1):

f[i] = f[i-1] + f[i-2]

return f[n]

```

- 题目链接：[[https://leetcode.cn/problems/house-robber/description/]]
- 思考：
	- 选择偷现在这家，则不能偷前一家；也可不选偷这家，则可偷前一家。

```python
class Solution:

def rob(self, nums: List[int]) -> int:

n = len(nums)

if n==1:

return nums[0]

dp = [0]*(n)

dp[0] = nums[0]

dp[1] = max(nums[0],nums[1])

for i in range(2,n):

dp[i] = max(dp[i-1],dp[i-2]+nums[i])

return dp[n-1]
```

## 3. 最大子数组和
- 题目链接：[[https://leetcode.cn/problems/maximum-subarray/description/]]
- 思考：
- 解题方法一：我们可以一边遍历数组计算前缀和，一边维护前缀和的最小值（相当于股票最低价格），用当前的前缀和（卖出价格）减去前缀和的最小值（买入价格），就得到了以当前元素结尾的子数组和的最大值（利润），用它来更新答案的最大值（最大利润）。
	- 请注意，由于题目要求子数组不能为空，应当先计算前缀和-最小前缀和，再更新最小前缀和。相当于不能在同一天买入股票又卖出股票。
```python
class Solution: def maxSubArray(self, nums: List[int]) -> int: 
min_sum=pre_sum = 0 
max_sum = -inf 
for num in nums: pre_sum += num 
max_sum = max(max_sum,pre_sum-min_sum) 
min_sum = min(min_sum,pre_sum) return max_sum
```
- 解题方法二：定义 f[i] 表示以 nums[i] 结尾的最大子数组和。分类讨论：
	- nums[i] 单独组成一个子数组，那么 f[i]=nums[i]。
	- nums[i] 和前面的子数组拼起来，也就是在以 nums[i−1] 结尾的最大子数组和之后添加 nums[i]，那么 f[i]=f[i−1]+nums[i]。
```python
class Solution: def maxSubArray(self, nums: List[int]) -> int: f = [0] * len(nums) f[0] = nums[0] for i in range(1, len(nums)): f[i] = max(f[i - 1], 0) + nums[i] return max(f) 
```
- 解题方法三：空间优化。由于计算 f[i] 只会用到 f[i−1]，不会用到更早的状态，所以可以用一个变量滚动计算。状态转移方程简化为：f=max(f,0)+nums[i] f 可以初始化成 0 或者任意负数。
```python
class Solution: def maxSubArray(self, nums: List[int]) -> int: ans = -inf # 注意答案可以是负数，不能初始化成 0 
f = 0 
for x in nums: f = max(f, 0) + x 
ans = max(ans, f) 
return ans
```

2. [K 次串联后最大子数组之和](https://leetcode.cn/problems/k-concatenation-maximum-sum/)
	思路：如数组元素和>0，则前缀和最大值在最后一个数组，最小值出现在第一个，中间元素只需计算元素和即可；<=0的情况下必然出现在子数组，只需将两数组进行拼接计算即可

	```python
class Solution:

def maxSubArray(self, arr: List[int]) -> int:

max_res = f =0

for i in range(len(arr)):

f = max(f,0) + arr[i]

max_res = max(max_res,f)

return max_res

def kConcatenationMaxSum(self, arr: List[int], k: int) -> int:

MOD = 10**9+7 # 如 数组元素和>0，则前缀和最大值在最后一个数组，最小值出现在第一个，中间数组只需计算元素和即可；<=0的情况下必然出现在子数组，仅需将两数组拼接计算即可

if k==1:

return self.maxSubArray(arr)

res = self.maxSubArray(arr+arr)

res += max(sum(arr),0)*(k-2)

return res % MOD
```
3. [环形子数组的最大和](https://leetcode.cn/problems/maximum-sum-circular-subarray/)
思考：
- 分类讨论：子数组没有跨过边界，此时计算的是最大非空子数组和；子数组跨过边界，此时计算的是数组总和-最小子数组和，要注意的是最小子数组和为整个数组和的时候，取最大子数组和。
```python
class Solution:

def maxSubarraySumCircular(self, nums: List[int]) -> int:

max_s = -inf

min_s = 0

max_f = min_f = 0

for num in nums:

max_f = max(max_f, 0) + num

max_s = max(max_s, max_f)

min_f = min(min_f, 0) + num

min_s = min(min_s, min_f)

if min_s == sum(nums):

return max_s

return max(max_s, sum(nums) - min_s)
```
4. [拼接数组的最大分数](https://leetcode.cn/problems/maximum-score-of-spliced-array/)

```python
class Solution:

def processNums(self, nums1: List[int], nums2: List[int]) -> int:

max_f = max_res = 0

for num1, num2 in zip(nums1, nums2):

max_f = max(max_f, 0) + num2 - num1

max_res = max(max_res, max_f)

return sum(nums1) + max_res

  

def maximumsSplicedArray(self, nums1: List[int], nums2: List[int]) -> int:

return max(self.processNums(nums1, nums2), self.processNums(nums2, nums1))
```

2. [统计异或值为给定值的路径数目](https://leetcode.cn/problems/count-paths-with-the-given-xor-value/)
思考：定义 dfs(i,j,k)为从点 0,0到点 i，j异或值为 k 的路径数即可
```python
class Solution:
    def countPathsWithXorValue(self, grid: List[List[int]], k: int) -> int:
        MOD = 1_000_000_007

        @cache
        def dfs(i: int, j: int, x: int) -> int: # 从点0,0,grid[0][0]到 m-1，n-1，k 的方案数
            if i < 0 or j < 0:
                return 0
            val = grid[i][j]
            if i == 0 and j == 0:
                return 1 if x == val else 0
            return (dfs(i, j - 1, x ^ val) + dfs(i - 1, j, x ^ val)) % MOD

        return dfs(len(grid) - 1, len(grid[0]) - 1, k)
```

4. [机器人可以获得的最大金币数](https://leetcode.cn/problems/maximum-amount-of-money-robot-can-earn/)
思考：本题相当于可以不选路径上的至多 2 个数，可以额外增加一个参数 k，定义 dfs(i,j,k) 表示从 (0,0) 走到 (i,j)，在可用感化次数为 k 的情况下，可以获得的最大金币数。用「选或不选」分类讨论：
选：dfs(i,j,k)=max(dfs(i−1,j,k),dfs(i,j−1,k))+coins[i][j]。
不选（感化）：如果 k>0 且 coins[i][j]<0，则可以不选，dfs(i,j,k)=max(dfs(i−1,j,k−1),dfs(i,j−1,k−1))。
两种情况取最大值。
```python
class Solution:

def maximumAmount(self, coins: List[List[int]]) -> int:

  

@cache

def dfs(i: int, j: int, k: int) -> int:

if i < 0 or j < 0:

return -inf

x = coins[i][j]

if i == 0 and j == 0:

return max(x, 0) if k else x

res = max(dfs(i - 1, j, k), dfs(i, j - 1, k)) + x # 选

if k and x < 0:

res = max(res, dfs(i - 1, j, k - 1), dfs(i, j - 1, k - 1)) # 不选

return res

  

return dfs(len(coins) - 1, len(coins[0]) - 1, 2)
```
## 5. 背包
### 01背包
每个物品只能选一次
递推表达式：
$f[i][j] = \begin{cases} f[i-1][j-x] & \text{if } j \geq x（选） \\ f[i-1][j] & \text{if } j < x（不选） \end{cases}$

```python
# 记忆化回溯型
class Solution:
    def canPartition(self, nums: List[int]) -> bool:

        @cache
        def dfs(i: int, j: int) -> int: # dfs(i,j)表示能否从 nums[0]到 nums[i]中选出一个和恰好为 j 的子序列
            if i < 0:
                return j == 0
            return j >= nums[i] and dfs(i - 1, j - nums[i]) or dfs(i - 1, j)

        s = sum(nums)
        return s % 2 == 0 and dfs(len(nums) - 1, s // 2)
```
 ```python
 # 二维数组式
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        s = sum(nums)
        if s % 2:
            return False
        s //= 2
        n = len(nums)
        f = [[False] * (s + 1) for _ in range(n + 1)]
        f[0][0] = True
        for i, x in enumerate(nums):
            for j in range(s + 1):
                f[i + 1][j] = j >= x and f[i][j - x] or f[i][j]
        return f[n][s]
```

### 分组背包
同一组内的物品至多/恰好选一个。
[掷骰子等于目标和的方法数](https://leetcode.cn/problems/number-of-dice-rolls-with-target-sum/)
```python
class Solution:

def numRollsToTarget(self, n: int, k: int, target: int) -> int:

# 定义模数

MOD = 10**9 + 7

  

# DP 数组初始化

f = [[[0] * (target + 1) for _ in range(k + 1)] for _ in range(n + 1)]

  

# 正确设置基础条件

# 用 0 个骰子，凑成和为 0 的方法有 1 种。

# 为了让 f[i][k][...] 的逻辑成立，我们需要将 f[0][k][0] 设为 1

f[0][k][0] = 1

  

# 三重循环

for i in range(n):

for j in range(1, k + 1):

for num in range(1, target + 1):

# 状态转移

if j > num:

f[i + 1][j][num] = f[i + 1][j - 1][num]

else:

# 每次加法后都取模

f[i + 1][j][num] = (f[i][k][num - j] + f[i + 1][j - 1][num]) % MOD

return f[n][k][target]

```
## 6.线性 DP
### 最长公共子序列
定义 f[i][j] 表示对 (s[:i],t[:j]) 的求解结果，选与不选的衡量结果。
```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        len1 = len(text1)
        len2 = len(text2)
        f = [[0] * (len2 + 1) for _ in range(len1 + 1)]
        for i in range(1, len1 + 1):
            for j in range(1, len2 + 1):
                if text1[i - 1] == text2[j - 1]:
                    f[i][j] = max(f[i - 1][j], f[i][j - 1], f[i - 1][j - 1] + 1)
                else:
                    f[i][j] = max(f[i - 1][j], f[i][j - 1])
        return f[len1][len2]
```
### 最长递增子序列
- 方法一：可以求原序列与排序后序列的最长公共子序列长度
- 方法二：因为递增序列以小数字为准，所以我们可以二分搜索数字的左边插入位置，如果为 len，则进行添加，否则进行替换。
- 方法三：逐个枚举
```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        g = [] # g[i]表示f[j] = i+1时，nums[j]的最小值，也就是长为 i+1 的上升子序列的末尾元素的最小值
        for x in nums:
            j = bisect_left(g,x)
            if j == len(g): # >=x的 g【j】不存在
                g.append(x)
            else:
                g[j]=x
        return len(g)
```
## 7.划分型DP
### 判定能否划分
一般定义 f[i] 表示长为 i 的前缀 a[:i] 能否划分。
枚举最后一个子数组的左端点 L，从 f[L] 转移到 f[i]，并考虑 a[L:i] 是否满足要求。
例题：[检查数组是否存在有效划分](https://leetcode.cn/problems/check-if-there-is-a-valid-partition-for-the-array/)
```python
class Solution:

def validPartition(self, nums: List[int]) -> bool:

n = len(nums)

dp = [True] + [False] * n # dp[i]表示前 i 位是否可以正确划分

for i in range(n+1):

if i >= 2 and nums[i - 2] == nums[i - 1]:

dp[i] = dp[i - 2] or dp[i]

if i >= 3:

is_three_equal = (nums[i - 3] == nums[i - 2] == nums[i - 1])

is_three_consecutive = (nums[i - 3] + 1 == nums[i - 2] and nums[i - 2] + 1 == nums[i - 1])

  

if is_three_equal or is_three_consecutive:

dp[i] = dp[i] or dp[i - 3]

return dp[n]
```
