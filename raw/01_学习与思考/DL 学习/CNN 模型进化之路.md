# 一、模型架构进化之路
## 1.1 图像分类领域模型架构演进
```mermaid
flowchart TD
A(AlexNet:5 个卷积层、3 个全连接层) --> B(VGG:使用 3x3 小卷积核堆叠三层替代 7x7 卷积核，池化层后尺寸减半通道翻倍)
B --> C(ResNet:残差连接机制，将浅层网络的输出连接到深层网络)
C --> D(VIT:图像->分块->线性投影->classtocken+位置编码->transformer->class token输出->分类头)
```
**趋势**：提升深度，转向 transformer。
## 1.2 目标检测领域模型架构演进
```mermaid
flowchart TD
A(R-CNN:候选区域生成->CNN提取特征->SVM分类->边界框回归) --> B(Faster R-CNN:主干网络提取特征->RPN生成候选区域->ROI池化提取特征->检测头进行分类)
B --> C(YOLO:端到端训练，全卷积网络，将图像划分为SxS网格，每个网格负责预测B个边界框、C 个类别概率)
```
**趋势**：多阶段候选框->端到端单阶段检测->锚框机制优化。
## 1.3 图像生成领域模型架构演进
```mermaid
flowchart TD
A(GAN:生成器输出尽可能接近的真实数据<->判别器输出概率值判断是否为真实数据) --> B(Style GAN:引入中间潜在空间、自适应实例归一化、注入噪声生成高质量数据)
B --> C(Diffusion:扩散过程：向图像加入噪声<->逆扩散过程：从纯噪声经神经网络去噪还原真实数据)
C --> D(Stable Diffusion: 文本->CLIP编码->交叉注意力->隐空间扩散->VAE解码->图像)
```
趋势：判别器-生成器对抗->风格解耦->扩散建模->潜在空间扩散。
## 1.4 图像语义分割领域模型架构演进
```mermaid
flowchart TD
A(FCN:全卷积网络,消除全连接层,实现像素级预测) --> B(U-Net：跳跃拼接，将编码器第 i 层直接拼接解码器对称第 i 层)
```
趋势：全连接层->全卷积网络->跳跃连接融合多尺度特征。
# 二、数据增强技术
| 阶段   | 技术演进                   | 模型              |
| ---- | ---------------------- | --------------- |
| 早期基础 | 旋转、裁剪、颜色抖动             | AlexNet         |
| 空间变形 | 弹性变形增强形变鲁棒性（尤其生物医学领域）  | U-Net           |
| 归一化  | 局部响应归一化(LRN)->批归一化(BN) | AlexNet->ResNet |
| 高级生成 | 使用生成模型合成训练数据           | StyleGAN/DDPM   |
# 三、激活函数和正则化
| 技术      | 意义                        | 代表模型     |
| ------- | ------------------------- | -------- |
| ReLU    | 解决 Sigmoid/Tanh 梯度消失，加速收敛 | AlexNet  |
| Dropout | 在全连接层随机失活，抑制过拟合           | AlexNet  |
| AdaIN   | 自适应实例归一化，实现风格注入           | StyleGAN |

# 四、损失函数优化
| 问题场景   | 解决方案                                    | 代表模型  |
| ------ | --------------------------------------- | ----- |
| 目标检测定位 | 平方误差->IOU 责任分配(平衡大小框误差)                 | YOLO  |
| 类别不平衡  | 加权损失函数(强调边界像素和稀有类别)                     | U-Net |
| 生成对抗训练 | 原始 GAN 损失->Wasserstein 举例/LSGAN（解决梯度消失） | 后续改进  |
| 扩散模型   | 变分下界优化->噪声预测均方误差                        | DDPM  |
