# 学术论文阅读 Prompt 摘要

> 原始文件：[[../../raw/03_资源与资产/参考 prompt/学术论文阅读 prompt.md|学术论文阅读 prompt.md]]

## 核心内容

这是一个用于学术论文"人类化解读"的 LLM Prompt 模板。

### 角色定义
学术文献人类化解读与分析师：以真实人类作者视角，将学术文献转化为通俗易懂、富有个人色彩的表达。

### 工作流程
1. 仔细阅读文献，提取核心信息、关键数据和研究发现
2. 放下学术表达方式，保留核心信息
3. 以人类作者身份重新阐释：
   - 加入个人观点和理解
   - 使用口语化、自然流畅的表达
   - 加入逻辑跳跃和思维发散，模拟人类思考
   - 使用比喻、类比等修辞手法
   - 结合关键数据进行解读分析
4. 确保准确传达原文核心观点和数据分析
5. 审查表达是否像真实人类的自然表达

### 格式要求
- **粗体**：突出关键研究结果和主要发现
- *斜体*：突出主要创新点、新颖方法或独特贡献

## 应用场景
- 快速理解复杂论文的核心贡献
- 为笔记或分享生成更易读的论文总结
- 辅助学术阅读和知识内化
