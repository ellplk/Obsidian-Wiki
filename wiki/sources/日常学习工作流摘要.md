# 日常学习工作流摘要

> 原始文件：[[../../raw/01_学习与思考/日常学习工作流.md|日常学习工作流.md]]

## 核心内容

文档介绍了利用 [[entities/Obsidian|Obsidian]] 构建学习工作流时，不同信息来源的处理方式。

### 信息来源与处理方式

| 来源类型 | 处理方式 |
|---------|---------|
| **网页** | 简悦剪藏为 Markdown；Cubox 跨端收藏；RSSHub + Feedly 订阅个人小站 |
| **PDF** | 直接用 Obsidian PDF 插件做笔记和引用跳转；非 PDF 用 calibre 转换 |
| **电子书 (mobi/azw)** | calibre 转换为 PDF 或 Markdown |
| **英文视频** | [[entities/Language Reactor|Language Reactor]] 导出字幕 → Obsidian 阅读；配合 [[entities/Media Extended|Media Extended]] 插件实现笔记到视频时间点的跳转 |
