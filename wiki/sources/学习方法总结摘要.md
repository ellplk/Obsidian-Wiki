# 学习方法总结摘要

> 原始文件：[[../../raw/01_学习与思考/学习方法总结.md|学习方法总结.md]]

## 核心内容

文档总结了"结构搭建 + 可视化理解 + 应用驱动 + 迭代刷题"的数据结构与算法学习路径。

### 四步学习法

**Step 1：选一本有趣且贴近实际的书籍**
- 推荐《数据结构与算法之美》
- 标准：通俗易懂、结构清晰、案例丰富、贴合业务

**Step 2：构建知识树，形成全局认知**
- 使用思维导图工具（XMind、MindNode）
- 覆盖：线性结构、树与图、查找排序、经典算法

**Step 3：模块化拆解，逐一精读与表达**
每章节输出包含：
1. 由来：为何诞生？解决什么问题？
2. 基本概念：核心定义与性质
3. 通用模板：代码 + 注释
4. 时空复杂度分析
5. 可视化演示（VisuAlgo、动画等）
6. 实际应用场景

建议以**系列博客**形式发表，输出倒逼输入。

**Step 4：刷题实战，串点成线**
- 初期：经典题型，识别算法匹配关系（LeetCode、牛客）
- 中期：题单驱动（"二叉树100题"、"动态规划50题"）
- 进阶：模拟面试、限时刷题
- 每题后总结：分类、思路差距、薄弱点
