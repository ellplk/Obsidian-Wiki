# 动态规划学习笔记摘要

> 原始文件：[[../../raw/01_学习与思考/数据结构与算法/动态规划.md|动态规划.md]]

## 核心内容

动态规划（DP）是通过分解重叠子问题来求解优化问题的方法。

### 适用条件
1. **最优子结构**：问题最优解包含子问题最优解
2. **重叠子问题**：子问题被重复计算，可用记忆化避免

### 核心步骤
| 要素 | 说明 |
|------|------|
| 状态定义 | 用 `dp[i]` 表示子问题解 |
| 状态转移 | 递推关系式 |
| 初始条件 | 边界值 |
| 递推顺序 | 自底向上或自顶向下 |
| 结果提取 | 从 dp 数组获取最终解 |

### 实现方式
- **自顶向下**：递归 + 备忘录（记忆化搜索）
- **自底向上**：循环 + 数组/表格（通常效率更高，可滚动数组优化）

### 决策模型
1. **选/不选模型**：`dp[i] = max(选当前项, 不选当前项)`
   - 应用：打家劫舍、0/1 背包
2. **多状态决策模型**：定义多个状态表示不同路径
   - 应用：股票买卖（含冷冻期）

### 经典题型覆盖
- 爬楼梯、打家劫舍、最大子数组和
- 网格图 DP、01 背包 / 完全背包 / 分组背包
- 线性 DP（LCS、LIS）
- 划分型 DP
