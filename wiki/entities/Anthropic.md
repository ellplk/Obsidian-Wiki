# Anthropic

> AI 安全公司，Claude 系列模型的开发方

---

## 基本信息

- **类型**：AI 研究与商业公司
- **成立**：2021 年（由前 OpenAI 研究人员创立）
- **核心产品**：Claude 系列大语言模型

## 主要产品

- **Claude**：大语言模型系列（Haiku / Sonnet / Opus）
- **[[entities/Claude Code|Claude Code]]**：AI 编程助手 CLI 工具
- **Anthropic API**：面向开发者的 LLM API

## 技术特点

- 宪法 AI（Constitutional AI）对齐方法
- MCP（Model Context Protocol）——标准化 AI 工具扩展协议
