# PyTorch

## 简介

PyTorch 是由 Meta AI 开发的开源机器学习框架，以其动态计算图和 Pythonic API 而闻名，是学术界和工业界最常用的深度学习框架之一。

## 核心特性
- **动态计算图**：调试方便，灵活性高
- ** eager execution**：直观易懂的执行模式
- **torch.nn**：丰富的神经网络层和损失函数
- **torchvision / torchaudio / torchtext**：多模态工具包
- **FSDP / DDP**：分布式训练支持
