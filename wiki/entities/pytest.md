# pytest

## 简介

pytest 是 Python 社区最流行的第三方测试框架，以简洁、Pythonic 的方式编写和组织测试。

## 核心特性
- ** Fixtures (`@pytest.fixture`)**：可重用设置代码，支持 function/class/module/session 多作用域
- **断言**：直接使用 Python `assert`，失败信息详细
- **自动发现**：自动查找 `test_*.py` / `*_test.py` 中的测试函数
- **插件化**：pytest-cov（覆盖率）、pytest-xdist（并发）等
