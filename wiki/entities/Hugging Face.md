# Hugging Face

## 简介

Hugging Face 是一家专注于机器学习和自然语言处理的公司，以其开源生态和模型仓库而闻名。

## 核心产品与服务
- **Transformers 库**：最流行的预训练模型调用库（BERT、GPT、T5 等）
- **Hugging Face Hub**：模型、数据集、Space 应用托管平台
- **Datasets 库**：高效的数据集加载和处理工具
- **Diffusers 库**：扩散模型（Stable Diffusion 等）的工具库
- **Accelerate / PEFT / TRL**：训练、微调、RLHF 工具链
