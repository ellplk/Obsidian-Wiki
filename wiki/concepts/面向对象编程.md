# 面向对象编程

## 定义

面向对象编程（Object-Oriented Programming, OOP）是一种以"对象"为基本单元的编程范式，将数据和行为封装在一起。

## 四大特性

| 特性 | 说明 |
|------|------|
| **封装** | 将数据和行为捆绑，隐藏内部实现细节 |
| **继承** | 子类复用父类的属性和方法，建立层次关系 |
| **多态** | 不同对象对同一消息做出不同响应 |
| **抽象** | 关注本质特征，忽略不重要的细节 |

## 历史演进
- **Simula** (1960s)：首个引入对象、类、继承的语言
- **Smalltalk** (1970s)：纯面向对象，一切皆对象
- **C++** (1980s)：在 C 基础上加入 OO 特性
- **Java** (1990s)：简化 OO，推动企业级普及
