# 数据结构与算法学习体系

## 概述

本知识库中的算法学习资料主要围绕"结构搭建 + 可视化理解 + 应用驱动 + 迭代刷题"的系统性路径展开。

## 学习方法论

来自 [[sources/学习方法总结摘要|学习方法总结]] 的四步法：

1. **选书**：兴趣驱动，选通俗易懂、案例丰富的书籍（如《数据结构与算法之美》）
2. **建图**：用思维导图建立全局认知框架
3. **拆解**：每章节按"由来-概念-模板-复杂度-可视化-应用"六步法精读
4. **刷题**：从经典题型 → 题单驱动 → 模拟面试，串点成线

## 核心算法主题

### 动态规划
- [[concepts/动态规划|动态规划]] 是知识库中算法部分最详尽的笔记
- 覆盖：核心思想、实现方式、决策模型、经典例题
- 细分：背包问题、LCS、LIS、网格图 DP、划分型 DP

### 数据结构实现
在 raw/ 中有多个 Java 实现文件：
- 链表、栈、队列
